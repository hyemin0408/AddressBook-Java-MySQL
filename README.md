# AddressBook-Java-MySQL

 ## 프로젝트 소개  
학교 과제로 제작한 Java Swing 기반 주소록 관리 프로그램입니다.  
MySQL과 JDBC를 사용하여 데이터베이스와 연동되는 CRUD 기능을 구현했습니다.

 ## 사용 기술  
Java / Swing / MySQL / JDBC

## 주요 기능  
- 주소록 추가 및 삭제  
- 이름 검색 기능  
- 정보 수정 기능  
- 전체 데이터 조회

## 구현 과정에서 배운 점  
JDBC를 활용한 데이터베이스 연결 방법과 PreparedStatement를 통한 안전한 SQL 처리 방식을 학습했습니다.
